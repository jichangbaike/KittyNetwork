## Kitty Network 使用教程

### 1. 注册 Kitty Network 账号

访问 Kitty Network 官网，注册账号并登录。


### 2. 购买套餐

根据自己的流量需求选择合适套餐：

- 日常网页浏览、聊天：AIR 套餐
- 视频观看、AI 工具访问：STANDARD 套餐
- 大流量下载、多设备使用：PREMIUM 套餐


### 3. 获取订阅地址

购买套餐后：

1. 登录用户中心
2. 进入订阅管理页面
3. 复制订阅链接


### 4. 导入客户端

支持常见客户端：

- Clash Meta
- Shadowrocket
- V2RayN
- V2RayNG


导入订阅链接后，等待节点同步即可使用。
